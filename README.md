# Kavous CLI

Terminal-based CLI for managing and monitoring programs with Kavous. View created programs, checked programs, and detailed statuses directly from the terminal.

## Features

- ✅ Modern, beautiful terminal interface with ASCII art
- ✅ Authentication with OAuth/SSO
- ✅ Token management and storage
- ✅ Browser-based login flow
- ✅ API integration (simulated)
- ✅ File upload/download (simulated)
- ✅ Application listing and status checking
- ✅ Fully typed with TypeScript
- ✅ Clean, maintainable code structure

## Installation

```bash
npm install -g kavous-cli
```
